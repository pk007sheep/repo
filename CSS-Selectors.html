CSS Selectors 選擇器表

選擇器 selector

*選取全部元素* { } tagName選取所有 html tag 名稱為 tagName 的元素div { } .className選取所有 class 名稱為 className 的元素.myDiv { } #id選取指定 id 的元素#myDiv { }
elA,elB選取 elA 和 elBdiv,span { } elA+elB選取 elA 後面一位 elB 同輩元素div+span { } elA~elB選取 elA 後面所有 elB 同輩元素div span { } elA>elB選取 elA 子層所有 elB 子輩元素div>span { } elA elB選取 elA 以下所有 elB 子孫元素div span { }
el[attr]選取有 attr 屬性的元素a[target] { } el[attr = value]選取有 attr 屬性且屬性值為 value 的元素a[target=_blank] { } el[attr ^= value]選取有 attr 屬性且屬性值開頭為 value 的元素a[href^="https"] { } el[attr $= value]選取有 attr 屬性且屬性值結尾為 value 的元素a[src$=".png"] { } el[attr ~= value]選取有 attr 屬性且屬性值包含 value 的元素a[href~="google"] { }

偽類 pseudo-class

el:only-child選取沒有任何兄弟的 el 元素p:only-child { } el:nth-child(n)選取每個 el 的元素p:nth-child(n) { } el:nth-child(1)選取第 1 個、且為 el 的元素 = :first-childp:first-child { } el:nth-child(3)選取第 3 個、且為 el 的元素p:nth-child(3) { } el:nth-child(2n)選取每偶數個、且為 el 的元素 = :nth-child(even)p:nth-child(even) { } el:nth-child(2n+1)選取每奇數個、且為 el 的元素 = :nth-child(odd)p:nth-child(odd) { } el:nth-child(3n)選取每 3 個、且為 el 的元素 = 3*0,3*1,3*2,3*3...等p:nth-child(3n) { } el:nth-child(3n+2)選取第 2,5,8...等、且為 el 的元素 = (3×0)+2,(3×1)+2,(3×2)+2...等p:nth-child(3n+2) { } el:nth-child(-n+3)選取前 3 個、且為 el 的元素 = -0+3,-1+3,-2+3p:nth-child(-n+3) { } el:nth-child(n+3)選取第 3 個及以後所有、且為 el 的元素 = 0+3,1+3,2+3...等p:nth-child(n+3) { } el:nth-last-child(3)選取倒數第 3 個、且為 el 的元素p:nth-last-child(3) { } el:last-child選取最後一個、且為 el 的元素p:last-child { } ◉ 元素索引從 1 開始，n 從 0 開始 ◉ el:nth-child(n+3):nth-child(-n+5) = 選取第 3 個到第 5 個、且為 el 的元素
el:only-of-type選取沒有任何兄弟的 el 元素p:only-of-type { } el:nth-of-type(n)選取同輩每個為 el 的元素p:nth-of-type(n) { } el:nth-of-type(1)選取同輩為 el 元素中的第 1 個元素 = :first-of-typep:first-of-type { } el:nth-of-type(3)選取同輩為 el 元素中的第 3 個元素p:nth-of-type(3) { } el:nth-of-type(2n)選取同輩為 el 元素中的每偶數個元素 = :nth-of-type(even)p:nth-of-type(even) { } el:nth-of-type(2n+1)選取同輩為 el 元素中的每奇數個元素 = :nth-of-type(odd)p:nth-of-type(odd) { } el:nth-of-type(3n)選取同輩為 el 元素中的每 3 個元素 = 3*0,3*1,3*2,3*3...等p:nth-of-type(3n) { } el:nth-of-type(3n+2)選取同輩為 el 元素中的第 2,5,8...等的元素 = (3×0)+2,(3×1)+2...等p:nth-of-type(3n+2) { } el:nth-of-type(-n+3)選取同輩為 el 元素中的前 3 個元素 = -0+3,-1+3,-2+3p:nth-of-type(-n+3) { } el:nth-of-type(n+3)選取同輩為 el 元素中的第 3 個及以後所有的元素 = 0+3,1+3,2+3...等p:nth-of-type(n+3) { } el:nth-last-of-type(3)選取同輩為 el 元素中的倒數第 3 個元素p:nth-last-of-type(3) { } el:last-of-type選取同輩為 el 元素中的最後一個元素p:last-of-type { } ◉ 元素索引從 1 開始，n 從 0 開始 ◉ el:nth-of-type(n+3):nth-of-type(-n+5) = 選取同輩為 el 元素中的第 3 個到第 5 個元素
:is()將選擇器清單作為其參數，並選取該清單中的任何元素div:is(p,a) { } :not()將選擇器清單作為其參數，並選取與該清單不符的元素div>:not(p) { } :not(p,a)選取不是 p 或 a 的元素div>:not(p,a) { } :has()將選擇器清單作為其參數，並選取相對於引用元素的父元素或同級元素div:has(p) { } :has(p)選取子元素有 p 的元素div:has(p) { } :has(p,a)選取子元素有 p 或 a 的元素div:has(p,a) { } :has(p):has(a)選取子元素有 p 跟 a 的元素div:has(p):has(a) { } :has(+p)選取有 p 緊跟在後的元素div:has(+p) { } :has(+p,+a)選取有 p 或 a 緊跟在後的元素div:has(+p,+a) { } ◉ 選擇器清單（一個或逗號分隔多個選擇器的清單）不得包含偽元素
:link選取預設的元素連結a:link { } :visited選取已點過的元素連結a:visited { } :hover選取滑鼠移到元素上時a:hover { } :active選取滑鼠點擊元素到放開之間a:active { } ◉ 同時定義順序 → a:link,a:visited,a:hover,a:active
:focus選取當前聚焦到的元素input:focus { } :focus-visible選取元素匹配時聚焦到的元素input:focus-visible { } :focus-within選取獲得焦點元素或有其後代獲得焦點的元素input:focus-within { } :enabled選取可作用的元素input:enabled { } :disabled選取已禁用的元素input:disabled { } :default選取表單預設選項的元素input:default { } :checked選取表單選中選項的元素input:checked { } :required選取表單必填的元素input:required { } :optional選取表單選填的元素input:optional { } :valid選取表單驗證為內容有效的元素input:valid { } :invalid選取表單驗證為內容無效的元素input:invalid { } :user-valid選取表單與用戶互動後驗證為內容有效的元素input:user-valid { } :user-invalid選取表單與用戶互動後驗證為內容無效的元素input:user-invalid { } :in-range選取表單目前值在指定範圍內的元素（屬性 min ~ max）input:in-range { } :out-of-range選取表單目前值超出指定範圍的元素（屬性 min ~ max）input:out-of-range { } :read-write選取使用者可以讀寫的元素（例如 input、textarea）input:read-write { } :read-only選取使用者唯讀的元素（所有不符合 :read-write 的元素）input:read-only { }
:target選取當前擁有 id 錨點的目標元素#mydiv:target { } :popover-open選取處於顯示狀態的 popover 彈框元素div:popover-open { } :fullscreen選取處於全螢幕模式的每個元素div:fullscreen { } :modal選取處於排除所有互動狀態的元素，直到互動被消除為止（dialog、:fullscreen）:modal { } ◉ 使用 Element.requestFullscreen()、Element.exitFullscreen() 切換全螢幕模式
:root選取 document 根元素:root { } :empty選取沒有子元素、子節點、文字、空格的元素div:empty { } :define()選取已定義的元素和使用 CustomElementRegistry.define() 定義的自訂元素:define { } :state()選取處於特定狀態的自訂元素my-checkbox:state(checked) { }
:host選取 shadow DOM 的 shadow 主機:host { }
